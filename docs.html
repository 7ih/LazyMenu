Buttons

There are currently two button types: main & misc

Button Creation

Use the Menu.CREATE.[button type](text, function) function to create a button.
Example: Menu.CREATE.main("Hi!", function() { alert("You clicked the button!") });

Button Configurations

To configure Main buttons, use Menu.main
To configure Misc buttons, use Menu.misc
To configure all buttons, use Menu.all

Width

Configure the width of a button type.
The default value is dependant on the string length and font size.
Example: Menu.main.width = 100;

Height

Configure the height of a button type.
The default value is dependant on the font size.
Example: Menu.misc.height = 50;

Font Size

Configure the font size of a button type.
The default value is 64 for Main, and 32 for Misc.
Example: Menu.all.fontSize = 50;

Top

Configure a button type's distance from the top of the canvas.
The default value is 50 for Main, and 32 for Misc.
Example: Menu.main.top = 20;

Left

Configure a button type's distance from the left of the canvas.
The default value is 50 for Main & Misc.
Example: Menu.misc.left = 30;

Margin

Configure a button type's distance between other buttons.
The default value is 10 for Main & Misc.
Example: Menu.all.margin = 20;

Left Padding

Configure the text's distance from the left edge of the button of a button type.
The default value is 5 for Main & Misc.
Example: Menu.main.paddingLeft = 10;

Font

Configure the text font of a button type.
The default value is Arial.
Example: Menu.misc.font = "Arial";

Text

Configure the text color of a button type.
The default value is black.
Example: Menu.all.text = ["green", "lime"];
Note: the value must be an array. The array's length must be 2 or less. If the array's length is more than 1, the color will switch for each button.

Background

Configure the background color of a button type.
The default value is dodgerBlue and red for Main, and lime for Misc.
Example: Menu.main.bg = ["blue", "orange"];
Note: the value must be an array. The array's length must be 2 or less. If the array's length is more than 1, the color will switch for each button.

Button Options

Button options are an optional object parameter when creating buttons.

removeEventListeners

If set to true, the click & hover event listeners will be removed. This is useful if the canvas will be reset with another page.
The default value is True for Main, and False for Misc.
Example: Menu.CREATE.main("Lorem", function(){ alert() }, {removeEventListeners: false})

centerText

If set to true, the text will be aligned in the center of the button.
The default value is false for Main and Misc.
Example: Menu.CREATE.main("ipsum", function(){ alert() }, {centerText: true});

Images

To display an image in the menu, use Menu.img.src = [image source].
The source can either be a local link, or absolute link.

Examples:
Relative Link: Menu.img.src = "image.jpg" //image.jpg is a local file
Absolute Link: Menu.img.src = "https://example.com/img/image.jpg"

Image Configuration

Top

Configure the image's distance from the top of the canvas.
The default value is 65.
Example: Menu.img.top = 50;

Left

Configure the image's distance from the left of the canvas.
The default value is 500.
Example: Menu.img.left = 200;

Functions

Menu.draw

Draw the menu after configuration.
Parameters: canvas element
Example: Menu.draw(canvas) // canvas = canvas element

Miscellaneous Functions

Menu.posOnButton

Detect if a position (cursor) is over another position (button).
Parameters: pos1 (cursor), pos2 (button), button configuration (width & height)
Example: Menu.draw(canvas) // canvas = canvas element

Menu.drawButton

Draw a button.
Parameters: pos, config, color
Example: Menu.drawButton(Menu.buttons.main, Menu.main, true);

Number.prototype.realPos

Caluclate the correct position of a cursor for accurate tracking.
Parameters: axis
Example: event.clientX.realPos("x");
