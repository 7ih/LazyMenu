Purpose

This is an API for the HTML canvas, written in Javascript. This should be used for canvas browser games/apps. This is especially useful for lazy developers who don't want to use any CSS (To clarify, there is no CSS used in LazyMenu). Honestly, this isn't very useful in terms of practicality or visual appeal.

Getting Started

First, you need to install LazyMenu. There are two ways you can do this:

  1. Directly download/copy the latest LazyMenu file from jsdeliver and add it as a JS file to your project. Remember to refer to it in the HTML body!
  2. Download the file via HTML by adding the following code to the body: <script src="https://cdn.jsdelivr.net/gh/7ih/LazyMenu@main/libs/LazyMenu.js"></script>
    3. Afterwards, you should have access to LazyMenu.

How to use

To use LazyMenu, make sure you have a canvas element. To use the API, you will need to use the Menu object. To create a button, use Menu.CREATE as shown below:

Menu.CREATE.main("Hello");

This will tell the API to create a Main button that says "Hello, World!".
That's cool and all, but what about other types of buttons?

In Lazymenu, there are two types of buttons: Main and Misc. To add a Misc button:

Menu.CREATE.misc("World!");

Then, you need to tell the API to actually draw the menu. You can achieve this by calling the Menu.draw() function, like this:

Menu.draw(canvas);

The canvas variable is, obviously, the canvas element.

If you would write all of that into a javascript file, the end result should look like this: 
const canvas = document.getElementById("canvas"); // canvas element

Menu.CREATE.main("Hello");
Menu.CREATE.misc("World!");

Menu.draw(canvas);

If the HTML was set up correctly, then the page should show two buttons. You can also view the output here. However, you can't interact with the buttons yet because they don't have a function attached to them. To add some interactivity, add a function as a second parameter when creating a button. For example: 
function clickHandler() {
  alert("You clicked the button!");   
}
        
Menu.CREATE.main("Click Me!", clickHandler)

Now when this button is clicked, it will fire the clickHandler function, which gives a window alert. You can view the output here.
Congratulations, you made a working button!

More information will be in the docs. For example, editing button positions, colors, options, etc.
